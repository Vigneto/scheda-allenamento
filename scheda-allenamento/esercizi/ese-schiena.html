6 esercizi per allenare la schiena

Guida rapida con spiegazione, set consigliati e immagini illustrative.

Lat machine avanti

Seduto, impugna la barra larga e tirala verso il petto controllando la salita e la discesa.

3-4 set · 8-12 ripetizioni

Rematore con bilanciere

Con bilanciere, busto inclinato e schiena dritta, tira il peso verso l’addome e ritorna controllando.

3-4 set · 6-12 ripetizioni

Pulldown inverso

Impugna la barra con presa inversa (palmi verso di te) e tira fino al petto mantenendo i gomiti vicino al corpo.

3 set · 10-15 ripetizioni

Rematore con manubrio

Appoggia un ginocchio su una panca, schiena neutra, tira il manubrio verso il fianco e ritorna lentamente.

3 set · 10-12 ripetizioni per lato

Hyperextension (estensioni lombari)

Sdraiati sulla panca per estensioni, piega lentamente il busto in avanti e ritorna finché il corpo è allineato.

3 set · 12-20 ripetizioni

Superman (a corpo libero)

Sdraiato prono, solleva simultaneamente braccia e gambe mantenendo la posizione per alcuni secondi.

3 set · 20-40 secondi

Creato per un allenamento schiena completo e accessibile.
