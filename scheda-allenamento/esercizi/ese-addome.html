6 esercizi per allenare l'addome

Guida rapida con spiegazione, set consigliati e GIF illustrative.

Crunch classico

Sdraiati supino, gambe piegate, mani dietro la testa o petto. Solleva le scapole contrarre gli addominali.

3 set · 12-20 ripetizioni

Bicycle crunch (bici)

Alterna gomito e ginocchio opposto, mantieni il core contratto e movimento controllato.

3 set · 30-50 secondi

Plank (sul gomito)

Mantieni il corpo in linea retta dalle spalle alle caviglie, contrai glutei e addominali.

3 set · 30-90 secondi

Leg raises (gambe dritte)

Sdraiato supino, solleva le gambe dritte fino a 90°, controlla la discesa.

3 set · 8-15 ripetizioni

Russian twist

Siediti con busto inclinato, ruota il busto a destra e sinistra mantenendo il core contratto.

3 set · 20-40 ripetizioni

Mountain climbers

In posizione di plank alto, porta alternativamente le ginocchia al petto con ritmo sostenuto.

3 set · 30-60 secondi
