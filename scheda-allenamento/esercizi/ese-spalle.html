6 esercizi per allenare le spalle

Guida rapida con spiegazione, set consigliati e immagini illustrative.

Shoulder press con manubri

Seduto con schiena dritta, spingi i manubri verso l’alto estendendo completamente le braccia, poi ritorna lentamente.

3-4 set · 8-12 ripetizioni

Alzate laterali

Con i manubri ai lati, solleva le braccia lateralmente fino alla linea delle spalle mantenendo i gomiti leggermente flessi.

3-4 set · 12-15 ripetizioni

Military press con bilanciere

In piedi, afferra il bilanciere all’altezza delle spalle e spingilo sopra la testa mantenendo il core contratto.

3-4 set · 6-10 ripetizioni

Alzate frontali

Con i manubri davanti alle cosce, solleva le braccia frontalmente fino all’altezza delle spalle e torna giù controllando.

3 set · 12-15 ripetizioni

Face pull al cavo

Con una corda al cavo alto, tira verso il viso separando le estremità e mantieni i gomiti alti, concentrandoti sui deltoidi posteriori.

3 set · 12-20 ripetizioni

Pike push-up

In posizione a V rovesciata, piega i gomiti portando la testa verso il suolo e spingi indietro. Ottimo a corpo libero per i deltoidi anteriori.

3 set · 8-15 ripetizioni

Creato per un allenamento spalle completo e accessibile.
